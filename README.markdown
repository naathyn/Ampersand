Getting Started
---------------

    $ git clone git@github.com:naathyn/Ampersand.git amp
    $ cd amp
    $ bundle install
    $ bundle exec rake db:setup
    $ bundle exec rake db:populate
    $ bundle exec rake db:test:prepare
